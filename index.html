MOTION EXP.

A prototype of a mobile web app for moviegoers. It was created from a personal issue I had with keeping track of the movies I saw, and be aware of special screenings happening in my town. It was a team project designed and shipped to production in under two weeks during Le Wagon's full time web development bootcamp.

Features:

  • Search for movies by title, director or cast
  • Get movies informations (credits, synopsis, trailer, screening times)
  • Keep track of movies by adding them to either watchlist or history list
  • Activate notifications on specific movies to be notified if ever a screening is programmed in a cinema near the user
  • Chat with friends within the app
  • Ruby on Rails
  • PostgreSQL
  • Javascript
  • Stimulus
  • Bootstrap
  • Figma
  • Heroku

E-CAR

A web application prototype that allows users to both rent electric cars and offer their own vehicles for rent. It was a team project developped in one a week.

Features:

  • Search for car by type and dates
  • Get car informations and price regarding the rental period
  • As a user, access account informations, and booking list (past and pending)
  • As a owner, access account informations, list of cars to rent and booking request received
  • As a owner, accept or reject a booking request received
  • Ruby on Rails
  • PostgreSQL
  • Javascript
  • Bootstrap
  • Cloudinary
  • Heroku
